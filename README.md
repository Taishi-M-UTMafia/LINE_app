# README
LINEを模して製作された簡易チャットアプリです。 ProgateでRuby on Railsを学んだ直後に作られました。 

製作者：Taishi Murakami  
製作時間：1週間  

-使用言語・バージョン
  -Ruby 2.5.1
  -Rails 5.2.0

-主な機能：  
  -メール認証によるユーザー登録機能  
  -ログイン/ログアウト機能  
  -登録情報の変更機能  
  -１対１でのチャット機能(メイン)  
 
-備考
  -ローカル環境での一作目なのでかろうじて動くだけのダメアプリです。テストもリファクタリングもしていません。悪しからず。
